# Lead Intake

## Overview

The intake pipeline handles external lead submissions via the REST API.

## Key concepts

Leads are submitted via `POST /api/v1/leads` with a valid API key.

### Validation

All leads go through schema validation, fraud scoring, and quality scoring before routing.
