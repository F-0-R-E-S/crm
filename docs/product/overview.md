# Обзор продукта

## Что такое GambChamp CRM

GambChamp CRM — B2B SaaS платформа дистрибуции лидов для крипто/форекс аффилейт-маркетинга. Платформа связывает аффилейтов (источники трафика) с брокерами (покупатели лидов), обеспечивая интеллектуальную маршрутизацию, антифрод-защиту и полную аналитику.

## Ключевой поток

```
Аффилейт → REST API → Антифрод → Маршрутизация → Брокер → Статус-трекинг → Аналитика
```

1. Аффилейт отправляет лид через REST API
2. Система валидирует и скорит лид (антифрод)
3. Routing Engine выбирает оптимального брокера по правилам
4. Лид доставляется на API брокера
5. Статусы синхронизируются через postback-и
6. Данные агрегируются для аналитики и P&L

## Целевая аудитория

| Сегмент | Описание | Потребности |
|---------|----------|-------------|
| **Баинг-сети** | Компании, покупающие и перепродающие лиды | Маршрутизация, капы, P&L |
| **Аффилейт-сети** | Платформы, объединяющие аффилейтов | Управление аффилейтами, антифрод |
| **Медиабайеры** | Индивидуальные покупатели трафика | Аналитика, автологин, Telegram |

## Стратегическое позиционирование

### Ценовой сегмент
**$399–$899/мес** — средний сегмент с вакуумом:
- Ниже: непрозрачные решения (CRM Mate ~$700, Elnopy ~$200)
- Выше: enterprise-переусложнённые (HyperOne $999+)

### 5 осей дифференциации

1. **Прозрачное ценообразование** — публичные цены, без setup fees (ни один конкурент не делает этого)
2. **15-минутный SLA поддержки** — vs отсутствие SLA у конкурентов
3. **Встроенный cloaking** — только CRM Mate имеет это; для нас это P0
4. **Гарантированный автологин SLA** — конкуренты не гарантируют; мы берём на себя SLA
5. **Telegram-first операции** — 17+ типов событий через Telegram-бота

### Рыночные метрики

| Метрика | Значение | Источник |

| TAM (2025) | ~$1.8 млрд | Глобальный форекс/крипто CRM рынок |

| SAM CAGR | 18–22% | Выше общерыночного (12–13%) |

## Ключевые рыночные тренды

1. **Консолидация** — конкуренты расширяются из крипто/форекс в gambling и nutra
2. **Антифрод как оружие** — shave-protection стал определяющим pain point
3. **Автологин как hygiene factor** — базовое требование, дифференциация в глубине пайплайна
4. **Telegram-first** — основной рабочий интерфейс для медиабайеров
5. **Прозрачное ценообразование** — рынок созрел, но никто не делает это публично
